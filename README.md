# 🌤️ App de Clima

Aplicação web que mostra informações do clima em tempo real utilizando a API da OpenWeather.

## 🚀 Funcionalidades

* Buscar cidade
* Mostrar temperatura atual
* Temperatura máxima e mínima
* Umidade do ar
* Velocidade do vento
* Ícone do clima

## 🛠️ Tecnologias

* HTML5
* CSS3
* JavaScript
* API OpenWeather

## 🌐 Como usar

1. Digite o nome de uma cidade
2. Veja as informações do clima em tempo real

## ⚠️ Observação

Este projeto utiliza uma API pública apenas para fins de estudo.
